"# CodeAlpha_Projects" 
1. 🃏 Flashcard Quiz App (Task 1)
A study tool that allows users to create, manage, and review flashcards.
- **Features:** 3D flip animations, full CRUD operations (add, edit, delete), category tagging, progress tracking, and a rating system (Hard/OK/Easy).
  2. ✦ Random Quote Generator (Task 2)
An inspiring app that generates random quotes from various categories.
- **Features:** Curated quotes across 6 categories, favorites saving system, copy-to-clipboard, native share API, and a Dark/Light mode toggle.
3. 💪 Fitness Tracker Pro (Task 3)
A comprehensive dashboard to track daily fitness activities and goals.
- **Features:** Log workouts with calorie estimation (based on duration/weight), weekly bar charts, animated daily goal rings, history filtering, and sidebar navigation.
Technologies Used
- **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
🚀 How to Run
Since these are pure frontend applications, there is no setup or installation required!
1. Clone this repository or download the ZIP file.
2. Open the root `index.html` file in any web browser to view the centralized Dashboard.
3. Click on any of the project cards to open that specific application.
